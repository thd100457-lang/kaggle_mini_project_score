# 시험 성적 향상 정책 분석 보고서


# 1. 프로젝트 개요 (Project Overview)

본 프로젝트는 **학생의 출석률과 학습 행동 데이터를 기반**으로
시험 성적에 영향을 미치는 핵심 요인을 분석하고,
**데이터 기반 교육 정책 설계 가능성을 검증하는 것**을 목표로 한다.

기존 교육 정책은 교사의 경험이나 정성적 판단에 의존하는 경우가 많았으나,
본 연구는 통계 분석과 데이터 시각화를 통해
정량적 근거를 확보하고 정책적 개입 가능성을 탐구하였다.


# 2. 연구 배경 (Background)

학생 간 학업 성취도 격차는 지속적으로 존재했다.

출석 관리, 학습 시간 통제 정책은 시행되고 있으나 정책 효과에 대한 데이터 기반 검증 부족

**핵심 문제**
성적 향상 정책은 존재하지만,
실제로 효과적인지에 대한 정량적 근거가 부족하다.

# 3. 연구 목적 (Objective)

본 연구는 다음 세 가지 목적을 가진다.

1) 시험 성적에 가장 큰 영향을 미치는 학습 행동 요인 식별
2) 출석률 및 학습 시간이 성적 향상에 미치는 영향 검증
3) 데이터 기반 정책 개입 전략 도출

# 4. 데이터 설명 (Dataset)
출처 : Kaggle Playground Series S6E1 (Synthetic Student Dataset)

| 변수명              | 설명             |
| ---------------- | -------------- |
| age              | 학생 나이          |
| gender           | 성별             |
| study_hours      | 주간 학습 시간       |
| class_attendance | 출석률            |
| sleep_hours      | 수면 시간          |
| sleep_quality    | 수면 질           |
| study_method     | 학습 방법          |
| facility_rating  | 시설 만족도         |
| exam_difficulty  | 시험 난이도         |
| exam_score       | 시험 점수 (Target) |

# 5. 분석 방법 (Methodology)
5.1 EDA (Exploratory Data Analysis)
- 성적 분포 히스토그램
- 출석률 분포 분석
- 학습 시간 분포 분석
- 산점도 및 회귀선 시각화

5.2 변수 가공
- 출석률 그룹화 (Low / Mid / High)
- 학습 시간 구간화
- 파생 변수 생성

5.3 통계 검정
- ANOVA (일원분산분석)
- Tukey HSD 사후 검정

5.4 정책 시뮬레이션
- 출석 위험군 → 중간 그룹 전환 시 성적 변화 추정
- 학습 시간 한계효과 분석

# 6. 주요 분석 결과 (Key Findings)
6.1 성적 격차 존재
- 성적 분포가 넓게 퍼져 있음
- 학생 간 변동성이 큼

6.2 출석률과 성적의 관계
- 출석률 증가 → 평균 성적 상승
- 출석률 High 그룹의 성적 하한선이 높음
- 통계적으로 유의미한 차이 확인 (p < 0.001)

6.3 학습 시간의 한계 효과
- 학습 시간 증가 시 성적 상승
- 일정 수준 이후 효과 감소
- 양보다 질의 중요성 확인

# 7. 정책 제언 (Policy Recommendations)
7.1 조기 개입형 출석 관리 정책
- 출석 위험군 자동 식별
- 조기 경보 시스템 도입
- Low → Mid 전환 시 가장 큰 성적 상승 기대

7.2 학습 총량 관리 정책
- 성적대별 권장 학습 시간 가이드 제공
- 과잉/부족 학습 동시 방지
- 학습 효율 모니터링

7.3 데이터 기반 학습 피드백 시스템
- 개인별 데이터 리포트 제공
- 출석률, 학습 시간, 예측 성적 시각화
- 자기주도 학습 강화
- 예시 메시지:
    “현재 학습 패턴을 조금만 조정하면 더 좋은 성과를 기대할 수 있습니다.”
